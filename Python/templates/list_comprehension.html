ডিকশনারি কম্প্রিহেনশন

squares = {x: x*x for x in range(6)}
print(squares)
[output]: Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

This code is equivalent to

squares = {}
for x in range(6):
squares[x] = x*x
[output]: Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16, 5: 25}
