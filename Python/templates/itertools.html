কারতেশিয়ান প্রডাক্ট

আমরা কারতেশিয়ান প্রডাক্ট এর নাম তো অবশ্যই শুনেছি | Python এর Itertool Library ব্যবহার করে খুব সহজেই কারতেশিয়ান প্রডাক্ট বের করতে পারি | নিচের উদাহরণটিতে দুইটি লিস্ট এর কারতেশিয়ান প্রডাক্ট বের করা হয়েছে |

from itertools import product
a=[int(x) for x in input().split()]
b=[int(x) for x in input().split()]
print(*list(product(a,b)))

[output]: (1, 3) (1, 4) (2, 3) (2, 4)

আমরা কারতেশিয়ান প্রডাক্ট এর নাম তো অবশ্যই শুনেছি | Python এর Itertool Library ব্যবহার করে খুব সহজেই কারতেশিয়ান প্রডাক্ট বের করতে পারি | নিচের উদাহরণটিতে দুইটি লিস্ট এর কারতেশিয়ান প্রডাক্ট বের করা হয়েছে |

from itertools import permutations
a, b = input().split()
[print(''.join(i)) for i in permutations(sorted(a), int(b))]
