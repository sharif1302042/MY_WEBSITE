লিস্ট অপারেশনস


পাইথনে সবচেয়ে পরিচিত একটা ডাটা টাইপ হচ্ছে লিস্ট | আসলে লিস্ট পাইথনের একটি বিল্ট-ইন ডাটা স্ট্রাকচার | সি, সি+ এ যাহাই অ্যারে তাহাই পাইথনে লিস্ট | তবে অ্যারের চাইতে এর কিছু গুরুত্বপূর্ণ ফিচার আছে | আজকে আমাদের আলোচনার বিষয়বস্তু লিস্ট ডাটা টাইপ |

লিস্ট এর দৈর্ঘ্য বের করাঃ

লিস্ট এর দৈর্ঘ্য বের করার জন্য len(list)মেথড বেব্যহার করা হয় | মেথডে লিস্টটি পাস করতে হয় অবশেষে তা লিস্ট এর দৈর্ঘ্য রিটার্ন করে |

a = [1, 2, 3, 4, 5]
print(len(a))
output:5

আইটেম যোগ করাঃ

লিস্টে কোন একটা উপাদান ইন্ডেক্স যোগ করার জন্য append()মেথড বেব্যহার করা হয় | মেথডে উপাদানটি পাস করতে হয়, উপদানটি লিস্ট এর শেষে যোগ হয় |

a = [1, 2, 3, 4, 5]
a.apped(10)
print(a)
output:[1, 2, 3, 4, 5,10]

নিরদৃষ্ট ইন্ডেক্সে আইটেম যোগ করাঃ

লিস্টে কোন নিরদৃষ্ট ইন্ডেক্সে আইটেম যোগ করার জন্য insert(index, element)মেথড বেব্যহার করা হয় | index হচ্ছে কোন ইন্ডেক্সে element/উপদানটি যোগ হবে | উপদানটি লিস্ট এর নিরদৃষ্ট ইন্ডেক্সে যোগ হবে এবং পূর্বের উপদানটি মুছে যাবে |

a = [1, 2, 3, 4, 5]
a.insert(3, 100)
print(a)
output:[1, 2, 3, 100, 5]

আইটেম গুনাগুনিঃ

লিস্ট এ কোন একটি আইটেম কতবার আছে তা হিসাব করার জন্য count(element)মেথড বেব্যহার করা হয় | যতবার উপাদানটি পাবে তা রিটার্ন করবেে আর না পেলে 0 রিটার্ন করবে |

a = [1, 2, 2, 3, 4, 5]
print(a.count(2))
output:2
print(a.count(10))
output:0

লিস্ট যোগ করাঃ

এক লিস্টের সাথে অন্য লিস্ট যোগ করার জন্য extend()মেথড বেব্যহার করা হয় | 2nd লিস্টের উপাদানগুলো 1st লিস্টের শেষে যোগ হবে, উল্লেখ্য 2nd লিস্ট অপরিবর্তিত থাকবে |

lst1 = [1, 2, 3, 4, 5]
lst2 = [10, 20, 30]
lst1.extend(lst2)
print(lst1)
output:[1, 2, 3, 4, 5,10,20,30]

অন্য একটি উপায়েও এই কাজটি করা যায়.....

lst1 = [1, 2, 3, 4, 5]
lst2 = [10, 20, 30]
lst1=lst1+lst2
print(lst1)
output:[1, 2, 3, 4, 5,10,20,30]

ইন্ডেক্স খোঁজাখূজিঃ

লিস্টে কোন একটা উপাদানের ইন্ডেক্স খুঁজে বের করার জন্য index()মেথড বেব্যহার করা হয় | মেথডে উপাদানটি পাস করতে হয়, উপদানটি পেলে তার ইন্ডেক্স রিটার্ন করে অন্যথায় ValueError থ্রো করে |

a = [1, 2, 3, 4, 5]
a.index(5)
output:4

a.index(7)
Traceback (most recent call last):
File "stdin", line 1, in module
ValueError: 'p' is not in list

List.reverse() মেথডের সাহায্যে লিস্ট উলটানোঃ

প্রত্যেক লিস্ট এর একটা বিল্ট ইন মেথড হল reverse() মেথড | যা কোন লিস্ট এর উপাদান গুলোকে শুধু উলটা করে দেখায় কিন্তু নতুন কোন লিস্ট তৈরি করেনা | শুধু লিস্ট টাকে মডিফাই করে |

mylist = [1, 2, 3, 4, 5]
mylist.reverse()
print(mylist)
newlist output: [5,4,2,3,1]

"[::-1]" এর মাধ্যমে লিস্ট reverse করাঃ

লিস্ট reverse করার আর একটা উপায় হল [::-1]. এটাও লিস্ট এর উপাদান গুলোকে শুধু উলটা করে দেখায় কিন্তু নতুন কোন লিস্ট তৈরি করেনা |

mylist = [1, 2, 3, 4, 5]
mylist[::-1]
print(mylist)
output:[5, 4, 3, 2, 1]

রিভার্স ইটারেটর এর মাধ্যমে লিস্ট রিভার্স করাঃ

রিভার্স ইটারেটরের মাধ্যমেও আমরা কোন লিস্ট কে রিভার্স করতে পারি | এটাও নতুন কোন লিস্ট তৈরিও করেনা পূর্বের লিস্ট এর কোন পরিবর্তনও করেনা , শুধুমাত্র ইটারেটর জেনারেট করে লিস্ট টা দেখায় |

mylist = [1, 2, 3, 4, 5]
for item in reversed(mylist):
....print(item)
output:
5
4
3
2
1
print(mylist)
output:[5, 4, 3, 2, 1]

দেখা যাচ্ছে পূর্বের লিস্ট এর কোন পরিবর্তন হয়নি |

লিস্টের উপাদান মুছে ফেলাঃ

লিস্টের কোন উপদান মুছে ফেলার জন্য তিনটি মেথড রয়েছে remove(),del(),pop().

remove() মেথড সরাসরি ভেলু নিয়ে কাজ করে | মেথডে ভেলু পাস করার পর যদি সে লিস্টে ওই ভেলুটি পায় তাহলে মুল লিস্ট থেকে ভেলুটি মুছে দেয় অন্যথায় ValueError থ্রো করে |

a = [0, 2, 3, 2]
a.remove(2)
....print(a)
output:[0, 3, 2]

a.remove(7)
Traceback (most recent call last):
File "stdin", line 1, in module
ValueError: list.remove(x): x not in list

del() মেথড ইন্ডেক্স এর ভিত্তিতে কাজ করে | মেথডে ভেলু পাস করার পর যদি সে লিস্টে ওই ভেলুটি পায় তাহলে মুল লিস্ট থেকে ভেলুটি মুছে দেয় অন্যথায় IndexError থ্রো করে |

a = [3, 2, 2, 1]
del a[1]
print(a)
output:[3, 2, 1]

del a[7]
Traceback (most recent call last):
File "stdin", line 1, in module
IndexError: list assignment index out of range

pop() ইন্ডেক্স এর ভিত্তিতে কাজ করে | মেথডে সরাসরি ইন্ডেক্স পাস করতে হয় |ইন্ডেক্স পাস করার পর যদি সে লিস্টে ওই ইন্ডেক্স পায় তাহলে মুল লিস্ট থেকে ইন্ডেক্সটির ভেলুটি মুছে দেয় এবং তা রিটার্ন করে , অন্যথায় IndexError থ্রো করে |

a = [3, 2, 4, 1]
print(pop(1))
output:2
print(a)
output:[3, 4, 1]

pop a[7]
Traceback (most recent call last):
File "stdin", line 1, in module
IndexError: pop index out of range
