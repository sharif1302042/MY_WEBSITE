Django ORM and QuerySets

Django ORM:
একটা কথা বলা হয় "Python এ সব কিছুই Object" | Object-Relation Mapper কে সংক্ষেপে OMR বলা হয়, যা বিভিন্ন রিলেশনাল ডাটাবেস যেমন SQLite, PostgreSQL, MySQL এর সাথে ইন্টারকানেক্ট করার জন্য Django এর একটা ডিফল্ট মেকানিসম |

class Post(models.Model):

author = models.ForeignKey('auth.User', on_delete=models.CASCADE)
title = models.CharField(max_length=200)
text = models.TextField()
created_date = models.DateTimeField(default=timezone.now)
published_date = models.DateTimeField(blank=True, null=True)

def publish(self):
self.published_date = timezone.now()
self.save()

def __str__(self):
return self.title

Database Access through Managers

Django ডিফল্টভাবে Objects নামে একটা Manager প্রত্যেকটা মডেল এ যোগ করে নেয় , এই Objects এর মাধ্যমেই আমরা Model এ Query করতে পারি |
Objects হচ্ছে django.db.models.manager.Manager class হচ্ছে এর ইন্সটেন্স মাত্র | এখন আমরা এই Objects manager এর কিছু গুরুত্বপূর্ণ মেথড নিয়ে আলোচনা করবো |
