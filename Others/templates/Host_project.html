WSGI ফাইল কনফিগার করুনঃ

WSGI configuation file এ ক্লিক করুন ইহা আপনাকে একটি কোড এডিটর এ নিয়ে যাবে | অনেকগুলো সেকশন দেখতে পাবেন, শুধুমাত্র Django সেকশন বাদে বাকি সব ডিলিট করুন |
আমি আমার প্রজেক্ট এর জন্য WSGI file নিচের মত কনফিগার করেছি , আপনিও একই ভাবে কনফিগার করুন | শুধুমাত্র Path টার ক্ষেত্রে একটু পরিবর্তন করুন আমি Path হিসাবে path = '/home/sharifthesimple/Dream-palace' দিয়েছি কেন দিয়েছি সেটা এর আগেই একবার ব্যাখ্যা করেছি এখানে তাই আর ব্যাখ্যা করার প্রয়োজন বোধ করছি না |

তারপর আপনার URL এ ক্লিক করুন | সব ঠিকঠাক থাকলে আশা করি আপনার প্রোজেক্ট অনলাইনে হোস্ট হয়েছে আলহামদুলিল্লাহ |
